快速开始

##1.下载源码

git clone git@github.com:spring-projects/spring-framework.git

相关 git知识学习廖雪峰官网;

##2.导入 IDE
进入源码根目录执行命令

./gradlew cleanIdea :spring-oxm:compileTestJava

导入Idea (File -> New -> Project from Existing Sources -> Navigate to directory -> 选择 build.gradle)

##3.导入依赖

./gradlew install

##4.构建

./gradlew build
